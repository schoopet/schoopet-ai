# Technology Stack

## Frontend (Web)
- **Framework/Bundler:** Vite
- **Languages:** Vanilla HTML, CSS, JavaScript
- **Deployment:** Firebase Hosting

## Backend (SMS Gateway)
- **Framework:** FastAPI
- **Language:** Python 3.11+
- **Hosting:** Google Cloud Run
- **SMS Provider:** Twilio

## AI Agents
- **Orchestration:** Google Agent Development Kit (ADK)
- **LLMs:** Google Vertex AI (Gemini 2.0 Flash, Gemini 1.5 Pro)
- **Memory:** Vertex AI Native Memory Bank

## Infrastructure & Services
- **Database (Session State):** Google Cloud Firestore
- **Secrets Management:** Google Cloud Secret Manager
- **Structured Data Store:** Google BigQuery
- **Deployment & Scaling:** Google Cloud Platform (GCP)

## Development & Quality Assurance
- **Testing:** Pytest, Pytest-asyncio
- **Workflow:** Conductor methodology
